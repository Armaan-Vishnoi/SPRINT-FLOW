# SprintFlow Backend

Features

- JWT Authentication
- Project Management
- Sprint Management
- Task Tracking
- Subtasks
- Dependencies
- Work Logs
- Notifications
- Socket.io Realtime Updates
- Cloudinary File Uploads

Tech Stack

- Node.js
- Express
- TypeScript
- MongoDB
- Socket.IO
- Cloudinary

Run Locally

npm install

npm run dev

Production

npm run build

npm start
